## Como ler

Cada ADR usa um formato leve:

- **Contexto**: problema ou ambiguidade.
- **Decisão**: opção adotada.
- **Alternativas consideradas**: opções avaliadas.
- **Consequências**: ganhos, custos e trade-offs.
